# Pset1 - Documentação da atividade

## Criação do Modelo Elmasri - SQL Power Architect

No programa **Power Architect** foi criado o diagrama das tabelas *funcionário*, *departamento*, *dependente*, *localizacoes_departamento*, *projeto*, e *trabalha_em* que vão ser inseridas no banco de dados, seguindo o modelo lógico Elmasri. 

> O **modelo lógico** consiste em determinar quais informações serão necessárias no banco de dados. 

Com a criação desse diagrama foram definidos os campos das tabelas, seus atributos e propriedades e ainda as chaves primárias, estrangeiras e alternativas (PK, FK e AK), além dos índices e relacionamentos.
